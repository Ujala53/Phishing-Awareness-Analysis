# Project 3: Phishing Awareness Analysis 🎣

## Introduction
In this project, I analyzed phishing messages to understand how attackers try to trick users into sharing sensitive information. This helps in building awareness about cyber threats.

## Objective
- Identify phishing attempts
- Recognize suspicious links and keywords
- Understand why phishing messages are dangerous

## Sample Phishing Message
Subject: Urgent! Your Bank Account Will Be Suspended

Dear User,  
Your account has been temporarily blocked due to suspicious activity.  
Please verify your account immediately:  
http://secure-bank-login.xyz/verify  

If not verified within 24 hours, your account will be permanently suspended.

## Analysis

### Suspicious Link
The link is not an official domain and uses a suspicious ".xyz" extension.

### Urgent Language
Words like "urgent" and "immediately" create panic.

### Generic Greeting
Uses "Dear User" instead of actual name.

### Threatening Tone
Mentions account suspension to pressure the user.

### Sensitive Action
Asks user to click a link and verify details.

## Red Flags
- Fake link
- Urgent tone
- Generic greeting
- Request for personal info
- Unusual domain

## Why This is Unsafe
This message is unsafe because it tricks users into sharing sensitive data through a fake link, which can lead to account hacking and data theft.

## Conclusion
This project helped me understand phishing attacks and how to identify them. It improved my awareness of cybersecurity threats.
